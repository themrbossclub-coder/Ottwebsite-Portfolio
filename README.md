# 🎬 OTT Streaming Website

A modern OTT (Over-The-Top) streaming platform that allows users to browse, watch, and manage movies and TV shows online. Built for performance, scalability, and an engaging user experience similar to platforms like Netflix, Prime Video, or Disney+.

---

## 🚀 Features

* 🎥 Browse movies and TV shows by category
* 🔍 Search functionality with real-time results
* 📺 Video streaming with adaptive playback
* 👤 User authentication (Sign up / Login)
* ❤️ Watchlist / Favorites
* ⭐ Ratings and reviews
* 📱 Fully responsive design (mobile, tablet, desktop)
* 🌓 Light/Dark mode support
* 🔐 Secure user data and authentication

---

## 🛠️ Tech Stack

* **Frontend:** React / Next.js / Vue
* **Backend:** Node.js / Express / Django
* **Database:** MongoDB / PostgreSQL / Firebase
* **Streaming:** HLS / DASH / Cloud-based CDN
* **Authentication:** JWT / OAuth / Firebase Auth
* **Styling:** Tailwind CSS / Bootstrap / SCSS

---

## 📦 Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/ott-platform.git
   ```

2. Navigate to the project directory:

   ```bash
   cd ott-platform
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Create a `.env` file and add required environment variables:

   ```env
   VITE_API_URL=your_backend_url
   VITE_AUTH_KEY=your_auth_key
   ```

5. Start the development server:

   ```bash
   npm run dev
   ```
